Пусть S - последовательность из N чисел пронумерованных подряд начиная с 1. Обозначим Si, Sj, Sk три элемента последовательности S, где i < j < k.

Определите в последовательности S три таких числа Si, Sj, Sk, что Si > Sj, Sk > Sj и значение выражения (Si - Sj) + (Sk - Sj) максимально.

В ответе укажите найденное максимальное значение выражения (Si - Sj) + (Sk - Sj).

Гарантируется, что в последовательности есть три числа Si, Sj, Sk, удовлетворяющие условию задачи.

Входные данные

Дано два входных файла (файл A и файл B), каждый из которых в первой строке содержит число N (5 < N < 10 000 000) - количество целых чисел. Каждая из следующих N строк содержит одно целое число, значение которого по модулю не превышает 1000

В ответе укажите два числа: сначала значение искомой величины для файла А, затем - для файла В.
